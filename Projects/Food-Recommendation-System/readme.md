# Food Recommendation System using RAG

A Python-based Food Recommendation System that uses **ChromaDB** for semantic similarity search and **Ollama LLM** to generate intelligent, conversational food recommendations using Retrieval-Augmented Generation (RAG).

## Features

* Semantic food search with ChromaDB
* Metadata-based filtering
* RAG-powered chatbot using Ollama
* Interactive, Advanced, and RAG search modes
* JSON-based food dataset

## Tech Stack

* Python
* ChromaDB
* Ollama (Llama 3.2)
* JSON

## Installation

```bash
git clone https://github.com/your-username/food-recommendation-system.git
cd food-recommendation-system
pip install -r requirements.txt
```

Pull the Ollama model:

```bash
ollama pull llama3.2
```

Start Ollama:

```bash
ollama serve
```

## Run

```bash
python rag_chatbot.py
```
